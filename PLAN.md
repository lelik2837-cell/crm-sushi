## Готово ✅

### Страница смены (`crm/templates/shift.html`)
Полностью переписана под макет `mockup_smena.html`.

- **Шапка страницы** — хлебные крошки, бейдж филиала, статус смены, кнопки «Сохранить» / «Закрыть смену»
- **Двухколоночный грид** (левая / правая колонки)
- **Блок выручки (левая колонка)**
  - Поля: Наличные, Безнал (карта), Онлайн, Доставка, Самовывоз
  - «Общая выручка» — автосумма (нал + безнал + онлайн), не редактируется вручную
  - «Итого нал в кассе» = нал − расходы нал − выплаченные ЗП
  - «Итого безнал» = только карта (онлайн не включается)
- **Блок расходов (левая колонка)** — добавление/удаление строк, выбор категории и типа оплаты
- **Кухня + Курьеры (правая колонка)** — грид-строки по ролям, группировка по заголовкам ролей
- **Строки сотрудников** — время начала/конца, часы (авторасчёт), км, заказы, итого, оплачено
- **Бонус/штраф** — раскрывающиеся строки под каждым сотрудником, цветная итоговая строка
- **Автосохранение** — fetch API при изменении любого поля (сотрудник, расход, выручка)
- **Модалка закрытия смены** и **модалка добавления сотрудника**
- **Sticky-панель** внизу страницы

### Исправленные баги
- `sqlite3.Row` + Jinja2 `sum` → заменено на JS-расчёт в `updateKassa()`
- Категория расхода в режиме просмотра — `selectattr` не работал с tuple, заменено на `{% for c, l in expense_categories %}`
- `sf-hours` для курьеров — был `<input>`, а не `<div>`, `calcStaffTotal()` проверяет `el.tagName`
- `autoSaveStaffRow` затирал имя/роль → добавлены `data-name` и `data-role` на строку
- Кеш шаблонов Flask → перезапуск сервера через `kill $(lsof -ti:5050)`

### Денежные поля
- Все денежные поля переведены на `type="text" inputmode="numeric"` с классом `money-field`
- Форматирование с разрядами: `150 000`, `1 250 000` (через `toLocaleString('ru-RU')`)
- При фокусе — чистые цифры; при уходе — с пробелами
- Стрелки «вверх/вниз» скрыты через CSS (`::-webkit-inner-spin-button`)
- Минус заблокирован через `keydown` + `input` listener

## Ключевые технические решения

| Решение | Причина |
|---|---|
| `type="text" inputmode="numeric"` для денег | `type="number"` не поддерживает форматирование с пробелами |
| `readNum(el)` — парсит через `replace(/\D/g, '')` | Корректно читает числа с пробелами-разделителями |
| Общая выручка = hidden input + display div | Нужно и отображать и отправлять в форме |
| Итого безнал = только карта | Онлайн-деньги не попадают в физическую кассу |
| `data-name` / `data-role` на div строки | Иначе автосохранение затирало имя/роль сотрудника |
| Jinja2 `namespace` для prev_role | Jinja2 не поддерживает переприсваивание переменных в цикле |

## Маршруты Flask (app.py)

| Маршрут | Назначение |
|---|---|
| `GET /shift/<id>` | Страница смены |
| `POST /shift/<id>/save-revenue` | Сохранение выручки |
| `POST /shift/<id>/save-expense` | Сохранение расхода |
| `POST /shift/<id>/delete-expense/<id>` | Удаление расхода |
| `POST /shift/<id>/save-staff` | Сохранение строки сотрудника |
| `POST /shift/<id>/delete-staff/<id>` | Удаление сотрудника из смены |
| `GET /api/employee/<id>` | Данные сотрудника (для модалки добавления) |
| `POST /shift/<id>/close` | Закрытие смены |
| `POST /shift/<id>/reopen` | Переоткрытие смены |
