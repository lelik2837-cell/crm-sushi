# CRM Суши — что сделано

## Сессия 2026-06-23

### 1. Детальная страница сотрудника из отчёта ЗП
- В таблице зарплатного отчёта имя сотрудника стало ссылкой (открывается в новой вкладке)
- Новая страница `/reports/employee/<id>` — все смены сотрудника за период
- Показывает: дату, филиал, должность, часы, базу/бонус/штраф, итого, статус выплаты
- Кнопка перехода на карточку каждой смены

### 2. Несколько должностей и ставок в карточке сотрудника
- Новая таблица `employee_roles` в БД
- В модальном окне редактирования сотрудника раздел «Доп. должности и ставки»
- Можно добавить любую должность с отдельной ставкой (для курьера — ₽/км и ₽/заказ)
- Удаление доп. должности крестиком

### 3. Удаление импортированных таблиц смен
- Новая таблица `import_batches` + колонка `import_batch_id` в `shifts`
- Каждый импорт файла Excel создаёт batch-запись
- На странице `/excel-import` внизу — история импортов с датой, файлом, кол-вом смен
- Кнопка «Удалить» — каскадно удаляет смены, расходы, записи ЗП и выплаты

### 4. Увольнение и восстановление сотрудников
- Новые колонки в `employees`: `is_fired`, `fired_at`, `fired_comment`
- В карточке сотрудника кнопка «Уволить» с полем причины
- Уволенные убраны из основного списка → отдельная таблица внизу страницы
- Кнопка «Восстановить» возвращает сотрудника в активные

### 5. Отчёт по движению наличных
- Новый маршрут `/report/cash-flow` в меню Отчёты
- По дням: утром в кассе, выручка нал, расходы нал, ФОТ выплачен, вечером факт в кассе
- Разворот дня — таблица по каждому филиалу с кнопкой перехода на смену
- Фильтры по периоду и филиалу
