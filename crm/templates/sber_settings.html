Как это работает
  1. Войдите в личный кабинет СберБизнес → Sber API → сгенерируйте Client Secret
  2. Введите Client ID 71154 и Client Secret → нажмите «Сохранить»
  3. Нажмите «Подключить СберБизнес» — откроется страница Сбербанка, войдите и разрешите доступ
  4. Для каждого счёта включите переключатель «Авто» — при открытии раздела «Банк» выписки будут загружаться автоматически за последние 7 дней
