# ТЗ — Страница "Смена" (заполнение администратором)

## Общее
Страница для администратора — ручное заполнение данных по текущей смене. Состоит из блоков, расположенных по сетке (не на всю ширину страницы).

---

## Блок 1: Выручка (левый верхний, занимает часть ширины страницы)

Слева в блоке (друг под другом, поля для ручного ввода):
- [ ] Количество заказов
- [ ] Наличные
- [ ] Безналичные
- [ ] Онлайн

Справа в этом же блоке — две категории:
- [ ] **Доставка**: выручка, заказы
- [ ] **Самовывоз**: выручка, заказы

Все поля заполняются вручную.

---

## Блок 2: Кухня (справа от блока "Выручка")

- [ ] Администратор — стандартно один человек (выбор из списка/фамилия)
- [ ] Кнопка "Добавить администратора" (если нужно больше одного)
- [ ] Повара — 2 строки по умолчанию, выбор фамилии из списка
- [ ] Кнопка "Добавить повара" (добавление доп. строк)
- [ ] Уборщица — стандартно одна, выбор фамилии из списка
- [ ] Кнопка "Добавить уборщицу" (если нужно больше одной)

---

## Блок 3: Курьеры (под блоком "Кухня")

Для каждого курьера (можно выбрать нескольких):
- [ ] Фамилия — выбор из списка
- [ ] Километраж
- [ ] Количество заказов
- [ ] Часы работы
- [ ] Доп. премия или штраф (сумма)
- [ ] Комментарий к премии/штрафу

---

## Блок 4: Расходы (под блоком "Выручка")

Список расходов, каждая строка:
- [ ] Категория: "Магазин" / "Аптека" / "Ремонт" (с выбором подкатегории ремонта из списка)
- [ ] Сумма
- [ ] Тип оплаты: Наличные / По карте
- [ ] Отметка "Гуляш" (галочка — занесено или нет)
- [ ] Кнопка добавления новой строки расхода

---

## Блок 5: Такси (под блоком "Расходы")

- [ ] Выбор сотрудника(ов), которых отправили на такси (из списка, можно несколько)
- [ ] Сумма
- [ ] Тип оплаты: По карте / Наличные
- [ ] Отметка "Гуляш" (галочка — занесено или нет)

---

## Блок 6: Плюсы в кассу (ниже блока "Такси")

- [ ] Выбор категории из списка: "Отработанное масло", "За сдачу рыбы", "Другое"
- [ ] Если "Другое" — свободное текстовое поле для описания
- [ ] Сумма
